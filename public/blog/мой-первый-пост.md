# Мой первый пост

Это мой первый пост в блоге! Здесь я расскажу о том, как создал эту систему блогов.

## Заголовок раздела

В этом разделе я могу писать обычный текст с **жирным** и _курсивным_ форматированием.

### Подзаголовок

- Список элементов
- Еще один элемент
- И третий элемент

Также можно использовать нумерованные списки:

1. Первый пункт
2. Второй пункт
3. Третий пункт

## Код

Можно вставлять код:

```javascript
function hello() {
  console.log("Привет, мир!");
}
```
