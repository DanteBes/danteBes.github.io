# Как добавлять изображения в блог

В этом посте я расскажу, как добавлять изображения в ваши посты блога.

## Способ 1: Изображение в папке blog/

Самый простой способ - поместить изображение в папку `blog/` рядом с вашим MD файлом. Например, если у вас есть файл `мой-пост.md`, вы можете добавить туда `image.png`.

Затем в Markdown используйте следующий синтаксис:

```markdown
![Описание изображения](/blog/image.png)
```

## Пример

Вот пример изображения:

![Пример изображения](/blog/image.png)

## Поддерживаемые форматы

Вы можете использовать следующие форматы изображений:
- PNG (`.png`)
- JPEG (`.jpg`, `.jpeg`)
- GIF (`.gif`)
- WebP (`.webp`)
- SVG (`.svg`)

## Важно

- Изображения автоматически копируются из папки `blog/` в `public/blog/` при запуске `npm run dev` или `npm run build`
- Всегда используйте абсолютные пути `/blog/image.png` для изображений в постах

## Заключение

Теперь вы знаете, как добавлять изображения в свои посты! Просто поместите файл изображения в папку `blog/` и используйте стандартный синтаксис Markdown для вставки изображений.
